# Contributing to AI Hypervisor Platform

First off, thank you for considering contributing to the AI Hypervisor Platform! It's people like you that make this tool great.

## Development Process

1. Fork the repo and create your branch from `main`.
2. Make sure you have read [docs/developer/onboarding.md](docs/developer/onboarding.md).
3. If you've added code that should be tested, add tests.
4. If you've changed APIs, update the documentation.
5. Ensure the test suite passes (`make test`).
6. Make sure your code lints (`make lint`).
7. Issue that pull request!

## Code of Conduct

By participating in this project, you are expected to uphold our [Code of Conduct](CODE_OF_CONDUCT.md).
